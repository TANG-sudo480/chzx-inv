# 新能源汽车产业链按图索骥

## 项目概述

静态 HTML 单页应用，展示某某区（湖州长兴）新能源汽车产业链全景图谱，包含 Leaflet 地图、多标签页交互、过滤器、鱼骨图分析等功能模块。

## 技术栈

- **运行时**: Node.js 24（用于预览和部署服务）
- **包管理器**: pnpm
- **核心依赖**: http-server（静态服务器）
- **前端**: 纯静态 HTML + CSS + JavaScript，Leaflet 地图库

## 目录结构

```
/workspace/projects/
├── index.html              # 主页面

├── .coze                   # Coze 项目配置
└── scripts/
    ├── coze-preview-build.sh
    ├── coze-preview-run.sh
    ├── coze-deploy-build.sh
    └── coze-deploy-run.sh
```

## 关键入口 / 核心模块

- `index.html`: 单页应用入口，包含所有业务逻辑
- `local_enterprise_data.js`: 全区企业数据（长合区经济发展局联系企业名单）
- `scripts/coze-*.sh`: Coze 预览和部署脚本

## 运行与预览

- **预览命令**: `bash scripts/coze-preview-run.sh`
- **预览端口**: 5000
- **预览验证**: `curl http://localhost:5000` 返回 HTTP 200

## 用户偏好与长期约束

- 纯静态项目，无需构建步骤
- 所有 Coze 脚本统一放在 `scripts/` 目录
- 脚本采用幂等设计，可重复执行
- http-server 使用 `-c-1` 参数禁用缓存

## 常见问题和预防

- 静态服务器必须在 5000 端口启动
- 脚本内部使用 `SCRIPT_DIR` 推导项目根目录，不依赖调用时 pwd

## 产业数据规范化要求

### 企业卡片数据字段规范
每个企业供应商条目应包含以下字段：
- `rank`: 排名序号（1-15）
- `name`: 企业名称
- `listed`: 上市状态（上交所/深交所/创业板/科创板/港股/美股/非上市）
- `marketCap`: 市值或规模描述
- `loc`: 地理位置（精确到市/区）
- `lat/lng`: 经纬度坐标
- `type`: 类型标识（local/200km/nation）
- `note`: 核心简介（主供客户、技术优势、关键信息）
- `price`: 参考价格区间
- `capacity`: 产能信息（如：10GWh、100万台/年等）
- `established`: 成立时间（如：2003年）
- `employees`: 员工规模（如：2000人）
- `revenue`: 年营收（如：50亿元）
- `tag`: 核心标签（如：宁德时代供应商、比亚迪战略合作等）

### 详情卡片展示布局
- 三栏布局：本地企业 | 200km内 | 全国头部
- 每栏信息分行展示：排名徽章 → 企业名称 → 上市+市值 → 地址 → 成立时间 → 产能 → 员工规模 → 营收 → 简介 → 价格 → 核心标签

### 距离分类
- **local（本地）**: 某某区本地企业
- **200km（周边200km）**: 某某区周边200km范围内的上市企业
- **nation（全国）**: 全国头部上市公司

### 分类数量要求
每个产业分类应包含 15 家企业：
- 约 5 家 local + 200km 企业
- 约 10 家全国头部上市公司

## 智能筛选功能

### 筛选维度
智能筛选基于 2024-2025 年真实融资和扩产动态，包括：

1. **业态类型**: 电池系统、电驱电机、智能驾驶、智能座舱、热管理、底盘传动、精密制造、核心材料、整车、eVTOL
2. **融资阶段**: IPO上市、IPO冲刺、战略融资、B轮以上、C轮巨额、扩产定增
3. **扩产动态**: 大规模扩产、中等扩产、小规模扩产、暂无扩产
4. **产业趋势**: 固态电池、800V高压、碳化硅SiC、大圆柱电池、智驾域控、线控底盘、空气悬架、动力电池回收
5. **技术壁垒**: 低壁垒(配套加工)、中壁垒(总成件)、高壁垒(核心部件)、极高壁垒(芯片/算法)
6. **配套距离**: 5km内、20km内、50km内、100km内、200km内

### 综合评分体系
评分权重：融资阶段30% + 扩产动态30% + 产业趋势25% + 配套距离15%
